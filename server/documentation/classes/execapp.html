spawnRemoteProcess

  • spawnRemoteProcess(timeout: number): Promise<object | Error>
    • Defined in controllers/app.invoke/exec.app.ts:144

    Spawns a process inside a container. Prepares the request with all the input data and application files requerid to run the application remotly. A Write stream is created to hold the response of the container as a .zip file. The content is then extracted, it's verified if any error was reported by the container and the returned promise is then resolved.

    Parameters

    • timeout: number

      Max. allowed time that the request to the containers has to be processed.
